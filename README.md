# webcli

A website written with Noir and ClojureScript.

## Usage

```bash
lein deps
lein run
```

## Usage light table:

$ lein light

$ java -jar launcher.jar

(load-file "src/webcli/server.clj")
(in-ns 'webcli.server)
(-main)
